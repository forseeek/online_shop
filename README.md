# Simple Python Online Shop
Online Shop browser app powered with Python
## How to Run
1. Install dependencies:
   ```bash
    pip install -r requirements.txt
   ```
2. Make sure the `templates` folder exist and are not empty.
3. Start the application:
   ```bash
   python run.py
   ```
4. Open your browser and go to [http://127.0.0.1:5000/](http://127.0.0.1:5000/)


## Project Structure
- `app/` — Main application code
- `templates/` — Templates for pages
- `static/` — Static files like CSS, images etc.
- `db/` — SQLite database file
- `config.json` — Configuration parameters
- `run.py` —  Main file that start an app
